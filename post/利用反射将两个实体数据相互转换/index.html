利用反射将两个实体数据相互转换

在实现客户的一个需求时，需要将ORM实体和Entity实体完成转换，并最终调用方法映射到数据库中。理论上最简单但是复杂的办法是进行遍历。这里使用了反射对数据实体进行自动映射：

创建ConvertModel方法，其中T1为源Entity，T2为要转换的Entity

public static List<T2> ConvertToNewModel<T1, T2>(List<T1> list)
        {
            var t2 = new List<T2>();
            foreach (var item in list)
            {
                T2 model = default(T2);
                model = Activator.CreateInstance<T2>();

                System.Reflection.PropertyInfo[] pi1 = typeof(T1).GetProperties();
                System.Reflection.PropertyInfo[] propertys =                        model.GetType().GetProperties();
                for (int i = 0; i < pi1.Count(); i++)
                {
                    propertys[i].SetValue(model, pi1[i].GetValue(item, null), null);
                }
                t2.Add(model);
            }
            return t2;
        }

然后在业务代码中调用此方法

                    var newlist = ConvertModel.ConvertToNewModel<I_DRC_DI_ADI_PATIENT_TREAT_INFO, I_WS_DI_PATIENT_TREAT_INFO>(list.RespContent);
